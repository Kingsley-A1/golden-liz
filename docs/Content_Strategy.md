# Golden Liz Content Strategy

## Objective

Use clear boutique copy to convince the client that a website is more than an online catalogue. It is a trust system, product presentation system, order collection system, and customer relationship system.

## Primary website message

Golden Liz helps women discover polished beauty and jewellery essentials through a clean, premium, mobile-first boutique experience.

## Client persuasion notes

The site should quietly answer the client’s hidden objections:

- “Will people use it?” → mobile-first UI, product browsing, WhatsApp support.
- “Will it help sales?” → clear CTAs, product cards, checkout flow.
- “Will it be hard to manage?” → admin dashboard preview and mock JSON structure.
- “Can customers trust it?” → authenticity, delivery, order tracking, packaging.
- “Is it better than Instagram only?” → organized categories, product detail pages, cart, lifecycle.

## Product description formula

Use this structure:

1. What it is.
2. What moment it helps with.
3. Why it feels premium or useful.

Example:

> A polished lip set for soft everyday glam, perfect for work, brunch, and evening looks.

## CTA rules

Use CTAs that match customer readiness:

- Browse: Shop Collection, View Lookbook
- Decide: Add to Bag, View Details
- Commit: Proceed to Checkout, Confirm Mock Order
- Post-purchase: Track Order, Shop More
