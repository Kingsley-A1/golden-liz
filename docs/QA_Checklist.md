# Golden Liz QA Checklist

## Mobile-first checks

- [ ] Header fits at 360px width.
- [ ] Hero heading is readable and not cramped.
- [ ] Product cards are easy to tap.
- [ ] Buttons have at least 44px tap height.
- [ ] Cart summary is readable on mobile.
- [ ] Checkout form fields are easy to complete.
- [ ] No horizontal scroll.

## Visual checks

- [ ] Gold is not overused.
- [ ] Product images are consistent in crop quality.
- [ ] Cards have consistent radius and spacing.
- [ ] Dark sections use cream/white text only.
- [ ] Typography hierarchy is clear.
- [ ] UI does not look like a generic template.

## Content checks

- [ ] Copy is short, clear, and feminine-premium.
- [ ] No cheap sales language.
- [ ] No fake real-payment claims.
- [ ] Product descriptions are helpful.
- [ ] CTA labels are consistent.

## Functional checks

- [ ] Home page loads.
- [ ] Products page loads.
- [ ] Every product detail page loads.
- [ ] Cart page loads.
- [ ] Checkout page loads.
- [ ] Confirmation page loads.
- [ ] Account page loads.
- [ ] Admin page loads.
- [ ] No broken image domains.
- [ ] No TypeScript errors.

## Client demo checks

- [ ] Can explain why website improves trust.
- [ ] Can show product browsing.
- [ ] Can show checkout lifecycle.
- [ ] Can show admin dashboard value.
- [ ] Can show future WhatsApp/payment integration path.
