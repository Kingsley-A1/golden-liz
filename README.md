# Golden Liz — Boutique Cosmetics & Jewellery Store

Golden Liz is a mobile-first ecommerce prototype for a boutique women-focused cosmetics and jewellery brand. It is designed as a persuasive client showcase: clean, premium, warm, trustworthy, and immediately testable with mock JSON data.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-style component structure
- Mock JSON first
- Vercel-ready deployment

## Product goal

Convince a boutique cosmetics and jewellery business owner that a professional website can help her:

- present products beautifully
- build trust beyond social media DMs
- collect orders in a cleaner flow
- showcase new arrivals and bestsellers
- reduce repeated customer questions
- prepare for future online payments, inventory, delivery, and customer lifecycle management

## Quick start

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```
